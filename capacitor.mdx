## Install via npm

Qonversion Plugin package is available on [npm](https://www.npmjs.com/package/@qonversion/capacitor-plugin).

<CodeGroup>
  ```bash bash
  npm install @qonversion/capacitor-plugin --save
  ```
</CodeGroup>
