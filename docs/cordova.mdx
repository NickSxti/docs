## Install via npm

Qonversion Plugin package is available on [npm](https://www.npmjs.com/package/cordova-plugin-qonversion).

<CodeGroup>
  ```bash bash
  npm install @qonversion/cordova-plugin --save
  ```
</CodeGroup>
