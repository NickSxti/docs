## Installation

To use Qonversion in your Flutter app, add qonversion [as a dependency in your pubspec.yaml file](https://flutter.io/platform-plugins/):

<CodeGroup>
  ```bash bash
  dependencies:
    qonversion_flutter: ^9.0.0
  ```
</CodeGroup>

Then install dependency

<CodeGroup>
  ```bash bash
  flutter pub get
  ```
</CodeGroup>
