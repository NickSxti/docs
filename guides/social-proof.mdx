## Star Rating Component

![](/guides/images/docs/04cf9ce481ba190aac8c558ba8b6fc71b111f2d3bb48c03343f1ab16785b13b1-star-rating.svg)

All review templates use the **Star Rating** custom component, which supports:

| Property        | Description                                   |
| --------------- | --------------------------------------------- |
| **Rating**      | Value from 0-5, supports decimals (e.g., 4.9) |
| **Max Stars**   | Maximum number of stars to display            |
| **Size**        | Star size in pixels                           |
| **Fill Color**  | Color for filled stars                        |
| **Empty Color** | Color for unfilled stars                      |
| **Gap**         | Spacing between stars in pixels               |

**Partial fill support:** A rating of 4.9 displays 4 fully filled stars and 1 star filled to 90%.

## Best Practices

* **Keep reviews authentic** — use real customer quotes when possible
* **Highlight key benefits** — select reviews that mention specific value propositions
* **Limit text length** — shorter reviews are more scannable on mobile
* **Use diverse testimonials** — showcase different use cases and user types
* **Position strategically** — place social proof near purchase decisions
* **Match your app's rating** — use accurate star ratings that reflect your actual store rating
* **Combine formats** — use a Review Line in the header and Review Column below for maximum impact
