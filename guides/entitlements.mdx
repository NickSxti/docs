Most applications have only one entitlement that provides premium access.

## Create Entitlement

1. Navigate to the [Entitlements & Products configuration section](https://dash.qonversion.io/entitlements). Tap the Create Entitlement button.

![](/images/docs/173cbc2-Screenshot_2024-02-06_at_16.54.09.png)

2. Fill in the entitlement details:

![](/images/docs/cf149ca-Screenshot_2024-02-06_at_16.55.29.png)

* Name: short description that helps everyone in your team understand what features the entitlement unlocks.
* Entitlement ID: a unique ID to use on the SDK and API sides.
* Associated products: list of products that grant cross-platform entitlement.
