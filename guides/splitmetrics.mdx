title: "SplitMetrics Acquire"

## Requirements

1. Before setting up the integration with SplitMetrics Acquire, ensure that you have configured integration with [Apple Search Ads](apple-search-ads) and can view events attributed to this media source on the dashboard.
2. Create [custom conversions](https://help.splitmetrics.com/en/articles/3551808-creating-custom-conversions-to-track-in-app-events) on SplitMetrics Acquire side for your in-app events.

## Configure Integration

1. Navigate to the integrations page in Qonversion and select [SplitMetrics Acquire](https://dash.qonversion.io/app/integration/create?name=split_metrics).
2. Find SplitMetrics Acquire Client ID [following this guide](https://help.splitmetrics.com/en/articles/3551823-how-to-find-my-client-id) (login to a platform is required) and provide it to Qonversion.

![](/guides/images/docs/a9c567dc934cf6c885d82b601331b67d0e540b5c9ac0af5554e25fbd0bf066d0-CleanShot_2024-08-22_at_18.19.482x.png)

3. Configure Global Event Preferences:

   * Send Revenue Properties: Leave this enabled to receive revenue values. Disable only if you do not require revenue data.
   * Send Sales as Proceed: Choose whether to send revenue values net of App Storesâ€™ commission (default) or gross.
   * Send Sandbox Events: Activate this toggle if you wish to receive sandbox events during testing.

![](/guides/images/docs/0887c1180f2b50e5596341ed6aef421dd16f0a6135bd9389c6a74c9b71239c8e-CleanShot_2024-08-22_at_18.26.582x.png)

4. Fine-Tune Event Settings: For each event, click More options to configure specific settings:

   * Enable Event: Toggle on to include the event in this integration. Turn it off if you do not need it.
   * Send Revenue: Enable or disable sending revenue for this particular event.

![](/guides/images/docs/f6cfc91325b905873f58cf6bff1492ed43be952a64d8e7c4014f7cf969147bf0-CleanShot_2024-08-22_at_18.28.062x.png)

5. Save Your Configuration: Ensure that all settings are as desired and click Add new integration+/ Save to apply.

<Check>
  ### Done

  Now Qonversion will start sending in-app purchases and subscriptions data to your SplitMetrics account.
</Check>

<Info>
  ### Notes

  1. You must repeat these steps for each app connected to SplitMetrics Acquire.
  2. Qonversion only supports sending In-App events from users attributed with Apple Search Ads. Install postbacks are not supported.
</Info>
